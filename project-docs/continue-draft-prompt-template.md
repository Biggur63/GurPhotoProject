# Continue Draft Prompt Template

Используйте этот шаблон, когда нужно быстро получить черновик статьи прямо в открытый `.md`-файл.

Замените все блоки в угловых скобках на свои данные.

```text
Нужен черновик статьи для сайта о фотографии.

Тема статьи:
<TITLE>

Краткое описание:
<DESCRIPTION>

Структура и обязательные блоки:
<OUTLINE>

Что важно раскрыть:
<KEY_POINTS>

Ограничения:
- не выдумывай факты
- не добавляй то, чего нет в заданной структуре
- пиши связный черновик на русском языке
- избегай воды, штампов и рекламной риторики
- держи популярную, но содержательную подачу

Технические требования:
- не добавляй front matter
- не добавляй заголовок статьи уровня H1
- начинай сразу с текста статьи
- используй подзаголовки уровня `##` и `###`, если это нужно по структуре

Контекст проекта:
<PROJECT_RULES_OR_NOTES>

Объём:
<TARGET_LENGTH>

Сделай цельный черновик статьи.
```

## Быстрый пример заполнения

```text
<TITLE> = Как использовать окно как основной источник света
<DESCRIPTION> = Черновик статьи для блога GurFoto
<OUTLINE> =
- Вход: почему окно остаётся рабочим источником света
- Разбор: направление света, расстояние, фон, экспозиция
- Практика: три коротких сценария съёмки
- Ошибки: контраст, фон, неверная точка съёмки
- Вывод
<KEY_POINTS> =
- показать, что окно не равно мягкий свет по умолчанию
- объяснить роль расстояния до окна
- не уходить в студийные схемы
<PROJECT_RULES_OR_NOTES> =
- ориентируйся на ai-rules.md, article-template.md и style-reference.md
- не используй рекламный тон
<TARGET_LENGTH> = 2200-2600 слов
```
