# Continue workflow for article drafts

Этот сценарий нужен для связки `Codex -> файл статьи -> Continue -> черновик -> редактура`.

## Что делает Codex

Когда вы пишете: `сделай публикацию по теме ...`, я могу:

1. Создать файл статьи в `content/blog/<slug>.md`.
2. Создать папку для будущих изображений в `assets/images/<slug>/`.
3. Оставить в файле маркеры для вставки черновика из Continue.
4. Подготовить для вас готовый промпт под конкретную тему.

По умолчанию этот сценарий имеет приоритет:
- если пользователь просит подготовить статью или публикацию по теме, сначала нужно готовить заготовку файла, папку изображений и prompt для Continue
- не нужно сразу писать полный текст статьи, если пользователь отдельно не просит именно готовую статью без этапа Continue
- варианты названия статьи не нужно поручать Continue или бесплатной модели; заголовок для frontmatter и prompt подбирает Codex

## Что делаете вы

После этого от вас нужны только эти действия:

1. Откройте созданный `.md`-файл в VS Code.
2. Поставьте курсор между `<!-- CONTINUE_DRAFT_START -->` и `<!-- CONTINUE_DRAFT_END -->`.
3. Откройте панель Continue.
4. Выберите модель `Draft Free Router` или `Mistral Free`.
5. Возьмите шаблон из `project-docs/continue-draft-prompt-template.md` и замените placeholders на тему статьи, пункты из контент-плана и заметки.
6. Отправьте запрос в Continue.
7. В ответе Continue нажмите `Insert at cursor` или `Apply`, чтобы текст попал прямо в открытый файл.

## Что получите в итоге

- Черновик сразу окажется в нужном файле статьи.
- Папка для изображений уже будет создана.
- После этого можно возвращаться ко мне для доработки структуры, смысла, SEO и финальной редактуры.

## Как выглядит файл после создания

В новом `.md` уже будут:

- front matter
- служебный блок с темой, slug и путём к папке изображений
- маркеры `CONTINUE_DRAFT_START` / `CONTINUE_DRAFT_END`

Черновик нужно вставлять между этими маркерами.

## Ежедневный сценарий

1. Вы даёте тему из контент-плана.
2. Я создаю заготовку статьи.
3. Я даю вам готовый промпт для Continue под эту тему.
4. Вы вставляете его в Continue и нажимаете `Insert at cursor`.
5. Мы доводим статью до публикации.
